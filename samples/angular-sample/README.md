## To generate an angular workspace with multiple applications in a frontend repository

Run `ng new {repo-name} --createApplication="false" --newProjectRoot=.`

## To generate an angular workspace with multiple applications in a frontend + backend repository

Run `cd {repo-name}`<br>
Run `ng new frontend --createApplication="false" --newProjectRoot=.`

## To generate an application inside an angular workspace

Run `cd frontend`<br>
Run `ng generate application {app-name}`
